# Week 3 Screenshots

These screenshots document the Excelerate Connect Week 3 functional Flutter prototype.

1. **Home — dynamic programs:** Home screen displaying program cards loaded from bundled JSON data.
2. **Program listing — JSON data:** Searchable and filterable program catalogue.
3. **Program details — top:** Selected program metadata, including category, deadline, duration, and delivery format.
4. **Program details — eligibility and actions:** Eligibility requirements and links to registration and feedback forms.
5. **Registration validation:** Required-field validation displayed after an invalid submission.
6. **Registration success:** Cleared registration form and local prototype success confirmation.
7. **Feedback success:** Cleared feedback form and local prototype success confirmation.
8. **Responsive layout:** Program listing tested using Chrome DevTools at an iPhone 12 Pro viewport.

The registration and feedback submissions are simulated locally. No information is sent to a server or stored persistently.
